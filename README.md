# 🔗 Web3 Python Toolkit

A collection of Python tools for interacting with the Ethereum blockchain in real-time.

## 🛠️ Tools Included

| File | Description |
|------|-------------|
| `main_file.py` | Connect to Ethereum & check wallet balance |
| `wallet_tracker.py` | Track ETH balance + USD value |
| `multi_wallet_tracker.py` | Monitor multiple wallets simultaneously |
| `etherscan_tracker.py` | Fetch transaction history via Etherscan API |
| `token_tracker.py` | Check ERC-20 token balances (USDT, USDC, SHIB) |
| `tx_tracker.py` | On-chain transaction scanner |
| `gas_tracker.py` | Real-time Ethereum gas fee monitor |

## ⚙️ Tech Stack

- Python 3.x
- web3.py
- Etherscan API v2
- CoinGecko / Binance API
- requests, datetime

## 🚀 Features

- ✅ Real-time ETH & token balance tracking
- ✅ Multi-wallet monitoring
- ✅ Transaction history lookup
- ✅ Live gas fee tracker (Slow / Normal / Fast)
- ✅ Auto-refresh every 30-60 seconds

## 📦 Installation

```bash
pip install web3 requests python-dotenv
```

## 🔑 Setup

Add your Etherscan API key:
```python
ETHERSCAN_API = "your_api_key_here"
```
